vJoy Library Setup Instructions
==============================

This folder should contain the vJoyInterfaceWrap.dll file required for the application to work.

To obtain this file:
1. Install vJoy from: http://vjoystick.sourceforge.net/
2. Copy vJoyInterfaceWrap.dll from: C:\Program Files\vJoy\x64\vJoyInterfaceWrap.dll
3. Paste it into this lib folder

The application will not build without this file.
